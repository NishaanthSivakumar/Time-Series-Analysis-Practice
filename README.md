# Time Series Analysis – Practice & Assignments

This repository contains my practice code and selected assignments for a university-level
**Time Series Analysis** module. The work focuses on both theoretical understanding and
hands-on implementation using **R** and modern time-series tools.

## Topics Covered
- Time series objects and tsibble framework
- Trend, seasonality, and decomposition
- Autocorrelation and partial autocorrelation
- AR, MA, ARMA, ARIMA models
- Exponential smoothing (ETS)
- Forecast evaluation and diagnostics

## Tools & Libraries
- R
- tidyverse
- fpp3
- tsibble
- fable
- feasts
- forecast

## Repository Structure
- `assignments/` – Coursework and structured analyses
- `practice/` – Practice scripts and exploratory models
- `data/` – Datasets used for analysis

> This repository is intended for learning and demonstration purposes only.
